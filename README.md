# Advanced House Price Prediction
This project involves prediction of house prices using well known Regression/Boosting Regressor models with implementation of cross-validation and evaluating performances using metrics such as: **MSE** (Mean_Squared_Error), **Mean_Squared_Log_Error**, **R2_Score**, etc. 

Link to the dataset: [dataset](https://www.kaggle.com/competitions/house-prices-advanced-regression-techniques/data)

Link to Scikit-Learn API Reference: [Scikit-Learn](https://scikit-learn.org/stable/modules/classes.html)

Link to Pandas API Reference: [Pandas](https://pandas.pydata.org/docs/reference/index.html)
